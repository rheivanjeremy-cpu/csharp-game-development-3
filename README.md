# 🎮 C# Basic Logic in Unity (IF Statement & Looping)  

A Unity project demonstrating fundamental C# programming concepts using conditional statements and loops.

## ✨ Features
- Pass or fail evaluation using if-else
- Grade classification (A, B, C, D)
- Even and odd number filtering using loops
- Console output using Debug.Log
  
## 🧠 Key Concepts
- If-else statement
- Nested conditions
- For loop
- Modulo operator (%)

## ▶️ How to Run
1. Open Unity Hub and create/open a project
2. Import or copy the scripts into the `Assets` folder
3. Create an empty GameObject in the scene
4. Attach the script(s) to the GameObject
5. Press Play and open the Console to see the output
   
## 🚀 How It Works
1. Evaluates a student's score to determine pass/fail status
2. Assigns a grade based on the score
3. Iterates numbers from 1–31 to filter even and odd values
4. Displays results in the Unity Console
   
## 🛠️ Tech Stack
- C#
- Unity Engine

## 📦 Requirements
- Unity (any recent version)
- Basic knowledge of C#
  
## 📁 Project Structure
- GradeEvaluator.cs → Handles score evaluation and grading
- EvenOddChecker.cs → Handles even and odd number filtering
